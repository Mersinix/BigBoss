ISSUE 1 – MY CATEGORIES DOES NOT SHOW SELECTED CATEGORIES
=========================================================

Current behavior:

Supplier
→ Categories
→ My Categories

When a supplier selects categories using:

Add Category Modal

the selected Category and its related SubCategories do NOT appear inside My Categories.

This is incorrect.

---

## EXPECTED BEHAVIOR

When the supplier selects a Category from:

Add Category Modal

the selected Category must immediately appear inside:

Supplier
→ Categories
→ My Categories

without requiring:

* Refresh
* Navigation away and back


The update should be immediate.

---

## SUBCATEGORY VISIBILITY

When the selected Category appears:

its related SubCategories must also appear correctly.

The supplier should be able to:

* View Category
* View SubCategories
* Select SubCategory

using the existing workflow.

---

## IMPORTANT

The recent filtering logic that only shows categories/subcategories related to supplier products appears to be preventing newly selected categories from appearing.

Investigate the implementation carefully.

Maintain BOTH requirements:

1. Categories selected through Add Category Modal must appear in My Categories immediately.

2. Categories/SubCategories that truly have no supplier association should still remain hidden when appropriate.

Do not remove the existing filtering logic entirely.

Fix the logic so both requirements work together correctly.
