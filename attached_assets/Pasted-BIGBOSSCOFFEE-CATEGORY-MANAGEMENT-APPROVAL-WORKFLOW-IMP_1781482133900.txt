## B. CATEGORY REQUESTS

Create a new section where suppliers can suggest new catalog values.

Reuse the same component structure and UI patterns from:

Admin > Categories > Category Management

BUT:

* Do NOT load admin catalog data into supplier tables.
* Supplier only sees records he created.
* Tables should initially be empty.
* This section is for suggestions only.

Supplier can create:

* Category
* Sub Category
* Brand
* Flavor
* Size

Every new record must automatically have:

status = PENDING
createdBySupplier = true
createdByUserId = supplierId
createdAt

While Pending:

* Record cannot be used as an approved platform catalog item.
* Record must display a Pending badge.
* Record remains frozen until admin review.

Supplier permissions:

Can:

* Create own suggestions
* Edit own Pending suggestions
* Delete own Pending suggestions

Cannot:

* Approve
* Activate
* Reject
