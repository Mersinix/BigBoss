Implement a complete B2B Promotion Management module for BigBossCoffee.

This feature must be supplier-driven and integrated across the entire marketplace (Supplier → Coffee Owner → Cart → Checkout → Orders). The system must be production-ready, scalable, and fully synchronized with the existing architecture.

## General Rules

* Every Supplier manages only their own promotions.
* Coffee Owners automatically receive eligible promotions.
* Promotions are applied automatically. No promo codes are required.
* Promotions must never affect other suppliers.
* If multiple suppliers sell the same admin product, each supplier's promotions are independent.
* Promotions must work correctly with the current marketplace logic, supplier products, packs, categories, cart, checkout, taxes, and order history.
* Existing functionality must not be broken.

---

# Business Rules

Promotions must never cross suppliers.

Each supplier controls only their own promotions.

Priority rules must be respected.

Expired promotions stop automatically.

Scheduled promotions activate automatically.

Inactive promotions are ignored.

Stacking rules must be respected.

The promotion engine should always choose the best valid promotion according to the configured rules.
