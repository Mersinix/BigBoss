# Goal

The user should no longer leave the Landing Page to log in.

The authentication experience should become:

1. Landing Page
2. User clicks **Connexion**
3. Authentication modal opens directly
4. User can either:

   * log in immediately
   * or switch to registration
5. During registration, the only behavior that changes is the **Type de compte** selection.

All existing authentication logic must continue working exactly as it does today.

Landing Page
